## What Is Aether?

Aether is an **AI-assisted litigation intelligence platform** that processes raw case data â€” pleadings, depositions, discovery productions, expert reports â€” into structured, AI-generated analysis for attorney review.

### Core Capabilities (17 Modules)

- **Case Chat Assistant** â€” Context-aware chat with RAG-powered document retrieval
- **War Room** â€” Strategic SWOT analysis with recommended actions
- **Red Team Audit** â€” AI-generated vulnerability matrix with severity scoring
- **Discovery Generator** â€” Draft interrogatories, RFPs, and RFAs
- **Document Drafting** â€” AI-generated draft text for motions and briefs
- **Citation Verification** â€” LLM-based plausibility check for legal citations
- **Hearing/Deposition Simulators** â€” Practice arguments with AI opposing counsel
- **Settlement Analysis** â€” AI-generated settlement scenario discussion
- **Privilege Scanner** â€” AI-based first-pass scan for protected content
- **Document Intelligence** â€” Text extraction and AI analysis of uploaded files
- **Semantic Search** â€” RAG pipeline with cosine similarity search
- **Executive Case Brief** â€” One-screen posture summary (pure data aggregation)
- **Timeline Tracking** â€” Chronological event management with calendar views
- **Case Management** â€” Per-case data separation with archiving
- **Intelligence Persistence** â€” Versioned analyses with input hash deduplication
- **Case Signals** â€” Pre-computed health metrics injected into AI prompts
- **Recalibration Engine** â€” Reactive staleness detection for analyses
- **Knowledge Graph** â€” Lightweight directed graph of case entity relationships

> **Important:** All AI output is decision-support material. Nothing generated by Aether constitutes legal advice. Attorney review is required before any use in legal proceedings.

## What It Does NOT Do

- **Not a lawyer.** All output requires attorney review.
- **No external legal research.** Citation verification is LLM-based plausibility checking, not database lookup.
- **No document export.** AI-generated drafts and analyses are viewable in the UI only â€” no DOCX or PDF download.
- **No page-level traceability.** Extracted events link to documents but not to specific pages.
- **Analysis versioning is backend-only.** Version chains exist in the database but UI doesn't expose version history navigation.
- **Limited access control.** Auth is header-based via identity proxy â€” minimal RBAC enforcement.
- **No job retries.** Failed document processing jobs must be re-triggered by re-uploading.
